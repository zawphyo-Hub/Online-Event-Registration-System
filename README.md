# Online-Event-Registration-System (Oxford Brookes University)

## Overview

The Online Event Registration System is a full-stack web application developed as part of the Computing Project module at Oxford Brookes University. The system enables users to create digital invitations, publish event registration pages, manage attendees, and verify participants through a built-in event verification feature.

The platform provides an efficient and user-friendly solution for event organizers to manage the entire event registration process, from invitation creation to attendee verification.

## Features

- Create and publish digital invitations and event registration pages.
- Generate unique shareable links for published events.
- Customize event pages using different event templates.
- Secure user authentication with Google Sign-In, OAuth2, and multi-factor authentication.
- Email confirmation service with event details and QR code verification.
- Interactive event location selection using Google Maps API.
- Dashboard for managing events and attendees.
- Built-in attendee verification functionality.
